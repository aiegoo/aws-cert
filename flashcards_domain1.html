12 Key Concepts
What AWS service automatically loads streaming data into S3, Redshift, or Elasticsearch?
Amazon Kinesis Data Firehose

Firehose is a fully managed service that buffers and batches data before delivery. It handles transformations via Lambda and requires minimal operational overhead.
What AWS data warehouse service provides petabyte-scale analytics?
Amazon Redshift

Redshift is a columnar MPP database optimized for OLAP. It supports Spectrum for querying S3 data and Concurrency Scaling for burst workloads.
What AWS service provides object storage with virtually unlimited scalability for data lakes?
Amazon S3

S3 is the foundation for data lakes, offering 11 9's of durability, lifecycle policies, versioning, and cost-effective storage classes (Standard, IA, Glacier). It supports batch and streaming ingestion.
What hybrid architecture combines data lake flexibility with data warehouse performance?
Lakehouse

Lakehouses use Delta Lake or Iceberg for ACID transactions on S3. They enable analytics and ML on the same platform with unified governance via Lake Formation.
What three dimensions characterize big data challenges?
Three Vs: Volume, Velocity, Variety

Volume = data scale (GB to PB), Velocity = speed (batch vs streaming), Variety = data types (structured, semi-structured, unstructured).
What AWS service captures and processes real-time streaming data with sub-second latency?
Amazon Kinesis Data Streams

Kinesis Data Streams provides real-time processing with custom applications. It retains data for 24 hours to 7 days, supports multiple consumers, and requires manual shard management.
What architecture stores raw data in its native format using schema-on-read?
Data Lake

Data lakes use S3 for storage, support unstructured/semi-structured data, and defer schema definition until query time. They enable exploratory analytics and ML workloads.
What AWS service provides serverless ETL and data cataloging for data lakes?
AWS Glue

Glue offers a data catalog, crawlers for schema discovery, and ETL jobs using Spark. It integrates with S3, Redshift, and RDS for unified metadata management.
What architecture optimizes structured data for BI and analytics using schema-on-write?
Data Warehouse

Data warehouses like Redshift require upfront ETL, use columnar storage, and optimize for complex SQL queries. They provide a single source of truth for reporting.
What process extracts, transforms, and loads data before storage?
ETL Pipeline

ETL pipelines transform data before loading into warehouses. AWS Glue provides serverless ETL with Python/Scala scripts and visual designers.
What approach defines data structure before ingestion?
Schema-on-Write

Used by data warehouses, schema-on-write enforces structure during ingestion. This ensures data quality but reduces flexibility for exploratory analysis.
What approach defers data structure definition until query time?
Schema-on-Read

Used by data lakes, schema-on-read stores raw data and applies structure during analysis. This enables flexibility but requires schema inference tools like Glue.
